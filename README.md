## How To Setup:

### 1. Clone repo:
`git clone git@github.com:avlogames/Pazaak.git`

### 2. Install Dependencies:
`cd pazaak && npm i`

### 3. Install Expo CLI
`npm i -g expo cli`

### 4. Run app.
`npm run ios`
